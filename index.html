    • Already added an ssh key to your computer and need to copy it to GitHub or GitLab?

    Use one of these commands to copy -


    Mac:
    tr -d '\n' < ~/.ssh/id_ed25519.pub | pbcopy

    Windows:
    cat ~/.ssh/id_ed25519.pub | clip
